Python学习记录--关于Tkinter Canvas(画布)的选项、方法、项目说明，以及一些示例。

元素（Item）

  • Arc（圆弧）
  • Bitmap（位图）
  • Image（图片）
  • Line（直线）
  • Oval（椭圆）
  • Polygon（多边形）
  • Rectangle（矩形）
  • Text（矩形）

Arc

  • 说明：绘制圆弧
  • 方法：create_arc(x0, y0, x1, y1, options...)
#示例
xy = 20, 20, 300, 180
canvas.create_arc(xy, start=0, extent=270, fill="red")
canvas.create_arc(xy, start=270, extent=60, fill="blue")
canvas.create_arc(xy, start=330, extent=30, fill="green")

xy，是圆弧左、上两个顶点的位置以及圆弧的X轴和Y轴的直径，如:xy = 20, 30, 300, 180，表明圆弧的左边顶点和画布左边界的距离为20像素，圆弧上边顶点和画布上边界的距离为30像素，圆弧X轴的直径为300像素，圆弧Y轴的直径为180像素

圆弧是以圆弧中心为基点，从X轴的3点钟方向开始，也就是X轴的0角度，逆向展开

圆弧属性（Options）

  • style
  • start
  • extent
  • fill
  • stipple
  • outline
  • outlinestipple
  • width
  • tags

style

  • Type：constant
  • 说明：圆弧的风格。其值有：pieslice, chord, arc
#示例
canvas.create_arc(xy, style = 'arc')

start

  • Type：angle
  • 说明：圆弧开始绘制时的起始角度。圆心为基点，正X轴为0角度，逆时针转动的角度。
#示例
canvas.create_arc(xy, start = 0)  #0角度开始
canvas.create_arc(xy, start = 90) #90度角开始，也就是正Y轴

extent

  • Type：angle
  • 说明：圆弧开始绘制时的结束角度。圆心为基点，正X轴为0角度，逆时针转动的角度。
#示例
canvas.create_arc(xy, start = 0, extent = 90)  #0角度开始，90度结束的一个扇形

fill

  • Type：color
  • 说明：圆弧内部的填充颜色
#示例
canvas.create_arc(xy, fill = 'red')

stipple

  • Type：bitmap
  • 待定

outline

  • Type：color
  • 说明：圆弧外部边框颜色。默认值为：black
#示例
canvas.create_arc(xy, outline = 'red')

outlinestipple

  • Type：bitmap
  • 待定

width

  • Type：distance
  • 说明：圆弧外部边框宽度，默认为1px
#示例
canvas.create_arc(xy, outline = 'red')

tags

  • Type：tuple
  • 待定

Bitmap

  • 说明：绘制位图
  • 方法：create_bitmap(x0, y0, options...)
#示例
canvas.create_bitmap(100, 100, bitmap="info", foreground="red")

位图属性（Options）

  • bitmap
  • anchor
  • foreground
  • background
  • tags

bitmap

  • Type：bitmap
  • 说明：位图名称，也可以用xbm文件。默认可用值为：error, gray75, gray50, gray25, gray12, hourglass, info, questhead, question, warning
#示例
canvas.create_bitmap(100, 100, bitmap = 'info', anchor="ne")

anchor

  • Type：constant
  • 说明：锚点。可用值：n, ne, e, se, s, sw, w, nw, center。默认为center
#示例
canvas.create_bitmap(100, 100, bitmap = 'info', anchor="ne")

foreground

  • Type：color
  • 说明：前景颜色
#示例
canvas.create_bitmap(100, 100, bitmap = 'info', foreground="red")

background

  • Type：color
  • 说明：背景颜色
#示例
canvas.create_bitmap(100, 100, bitmap = 'info', background="red")

tags

  • Type：tuple
  • 待定

Image

  • 说明：绘制图片
  • 方法：create_image(x0, y0, options...)。x0,y0为图片左上角坐标
#示例
photo = PhotoImage(file = 'eg.gif')
canvas.create_image(10, 10, image = photo)

图片属性（Options）

  • image
  • anchor
  • tags

image

  • Type：image
  • 说明：图片。需使用PhotoImage进行指定
#示例
photo = PhotoImage(file = 'eg.gif')
canvas.create_image(10, 10, image = photo)

anchor

  • Type：constant
  • 说明：锚点。可用值：n, ne, e, se, s, sw, w, nw, center。默认为center
#示例
canvas.create_image(10, 10, image = photo)

tags

  • Type：tuple

Line

  • 说明：绘制直线（折线）
  • 方法：create_line(x0, y0, x1, y1, ..., xn, yn, options...)
#示例
canvas.create_line(0, 0, 10, 10, fill = 'red')

直线属性（Options）

  • width
  • fill
  • stipple
  • arrow
  • arrowshape
  • capstyle
  • joinstyle
  • smooth
  • splinesteps
  • tags

width

  • Type：distance
  • 说明：线条宽度
#示例
canvas.create_line(0, 0, 100, 100, width =5)

fill

  • Type：color
  • 说明：线条颜色
#示例
canvas.create_line(0, 0, 100, 100, fill = 'red')
canvas.create_line(0, 0, 100, 100, fill = '#ff0000')

stipple

  • Type：bitmap
  • 说明：线条纹路。典型值有：gray75, gray50, gray25, gray12
#示例
canvas.create_line(0, 0, 100, 100, stipple = 'gray75', width =5)

arrow

  • Type：constant
  • 说明：带箭头的直线。值有3个，first, last, both，代表箭头位置：开始，最后，两边都有
#示例
canvas.create_line(0, 0, 100, 100, arrow = 'both', width =5)

arrowshape

  • Type：3-tuple
  • 说明：箭头形状。值是一个三元元组，默认值为：(8, 10, 3)
#示例
canvas.create_line(0, 0, 100, 100, arrowshape = (10, 10, 10), width =5)

capstyle

  • Type：constant
  • 说明：线条两头形状。需宽线，细线看不到效果。值有：buit, projecting, round。默认为buit。
#示例
canvas.create_line(0, 0, 100, 100, capstyle = 'round', width =5)

joinstyle

  • Type：constant
  • 说明：折线转折位置形状。需宽线，细线看不到效果。值有：bevel, miter, round。默认为round圆角
#示例
canvas.create_line(0, 0, 100, 100, 50, 20 capstyle = 'round', width =5)

smooth

  • Type：flag
  • 说明：折线转折位置启用平滑。值为bool，不为0或False时，启用平滑反之不启用
#示例
canvas.create_line(0, 0, 100, 100, 50, 20 smooth = True, width =5)

splinesteps

  • Type：int
  • 待定

tags

  • Type：tuple
  • 待定

Text
